Technical Info

If for some reason you're interested in the technologies I've used for this project here they are. This list is likely to keep on going the more I expand on the project.

Developing

  • The app itself - written in unity using C#
  • Users, Settings and Recovery Codes - managed by microservices written in js/ts mainly with express
  • Email Sending - handled by a C# .Net web service
  • Records - the records service, that mainly serves this website is written in java with spring, and unlike the other microservices that use rest, this one exposes a graphql api
  • This website - written in angular 16 and typescript

Deployments

  • The unity app is just built locally and then the apk is distributed in the Itch.IO Page
  • This website(and probably any other one I host) is deployed using Netlify
  • The microservices are mostly deployed using Render, besides one that is covered using the free plan at Cyclic
    ps. everytime loading takes a while, be it in app or when loading the table, blame it on Render
